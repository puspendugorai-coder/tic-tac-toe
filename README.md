## рҹҺЁ Design Highlights

The UI is inspired by a deep-space aesthetic:

- **Color palette** вҖ” deep navy backgrounds, cyan for Player 1, pink for Player 2, gold for draws and status
- **Starfield** вҖ” 90 drifting particles rendered on an HTML5 Canvas
- **Nebulae** вҖ” three slow-breathing ellipses layered behind the game board
- **Win flash** вҖ” winning cells pulse green with a highlight border
